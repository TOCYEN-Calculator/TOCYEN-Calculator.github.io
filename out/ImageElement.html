Name Type Description
path string The path of the image. Image won't be loaded immediantly.
position Vector | None | Number VECTOR: The relative position of the text's center on the canvas. NONE: Automatically Aligner.GetNextPosition(). Number: The padding of Aligner.GetNextPosition().
